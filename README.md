#API REST desenvolvida em java com Spring Boot para gerenciamento de tarefas (TO-DO List)

O projeto permite criar, listar, atualizar e remover tarefas, servindo como uma aplicação prática para estudo de desenvolvimento backend,
arquitetura em camadas e integração com banco de dados.

Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento backend, incluindo criação de APIs REST,
persistência de dados e boas práticas de organização de código.
